Gabby Notepad
うるさいメモ帳（英語専用）


## これは何？
Windows付属のメモ帳もどきです。
英語の文章を入力すると、単語毎に喋ります。

## 用途は？
- 英語学習に、文章を写経しましょう。


## 動作環境
- おそらく Windows 7 以降  
- .NET Framework 4 が利用可能であること

## 注意事項
日本語音声を選択すれば日本語の読み上げも可能ですが、  
単語の区切りや文末は認識していません。
