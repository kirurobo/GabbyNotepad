Gabby Notepad
うるさいメモ帳（英語専用）

Ver.1.0.0 2010/03/22 Ru--en

http://r13n.spaces.live.com/
ru-en@live.jp


○これは何？
Windows付属のメモ帳もどきです。
英語の文章を入力すると、1単語毎に喋ります。

○動作環境
.NET Framework 3.5 が必要です。

○注意事項
日本語は、たぶん喋れません。
（Windowsには標準で英語しか音声合成がないので。）

○用途は？
・英語の学習に。
・画面を見ず、紙を見ながら入力するときに。

○使用条件
配布や使用はご自由にどうぞ。
ほとんど、Windows 7 SDK のサンプル使っただけです。
このプログラムの使用に際して生じた問題に対し一切責任は負いません。
