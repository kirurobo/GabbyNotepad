Gabby Notepad
うるさいメモ帳（英語専用）

Ver.1.0.0 2010/03/22 Ru--en
Ver.1.1.0 2019/09/13 Kirurobo


○これは何？
Windows付属のメモ帳もどきです。
英語の文章を入力すると、1単語毎に喋ります。

○動作環境
.NET Framework 4 が必要です。

○注意事項
日本語音声を選択すれば日本語の読み上げも可能ですが、
単語の区切りや文末は認識していません。

○用途は？
・英語の学習に。
・画面を見ず、紙を見ながら入力するときに。

○使用条件
配布や使用はご自由にどうぞ。
このプログラムの使用に際して生じた問題に対し一切責任は負いません。
